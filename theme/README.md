# Theme

`BTA_Light.json` — custom Power BI theme for the Bank Transaction Analysis dashboard.

## Palette

| Role | Hex |
|---|---|
| Primary blue | `#378ADD` |
| Alert red | `#C0392B` |
| Light blue (series) | `#B5D4F4` |
| Neutral | `#D3D1C7` |
| Text secondary | `#6B7280` |
| Text primary | `#1F2A44` |
| Background | `#EAECF0` |

## How to apply

Power BI Desktop → **View → Themes → Browse for themes** → select `BTA_Light.json`.
